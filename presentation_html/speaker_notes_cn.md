# 激光选通文献分享讲稿

## 第1页：题目页

各位老师同学好，我今天汇报的主题是“基于激光选通图像序列的真假目标识别”。  

我这次主要分享三篇文献，它们分别从三个角度支撑我的研究：第一篇是激光选通成像的物理机制和距离精度模型，第二篇是三张选通图像如何通过网络形成深度先验，第三篇是 gated camera 在复杂感知任务中的多模态融合应用。

所以我今天不是单独讲某一篇论文，而是围绕我的课题主线，把这三篇文献串起来：激光选通图像为什么包含深度信息，这些信息如何被网络利用，以及它对真假目标识别有什么启发。

## 第2页：研究问题

我的研究问题是：如何利用激光选通图像序列区分真实三维目标和平面假目标。

如果只看普通二维图像，平面假目标只要有完整的外观纹理和轮廓，就可能和真实目标非常相似。但是激光选通图像不只是普通照片，它是通过控制相机的选通时间，只接收某一距离范围内返回的激光回波。

因此，对于真实三维目标，不同部位处在不同深度，三个选通门中的结构和亮度分布应该会发生变化。而对于平面假目标，整个反射图基本处在同一个深度平面上，所以不同 gate 中更可能表现为完整轮廓整体变暗，甚至消失，而不是出现真实三维结构切片。

这也是我后面数据集仿真和网络设计的核心依据。

## 第3页：文献1，选通成像的物理基础

第一篇文献是关于三维激光选通强度相关成像的距离精度预测模型。

这页图展示了激光选通成像的基本时空模型。系统发射激光脉冲，目标反射后，相机只在特定时间窗口打开，所以它接收到的是某一距离范围内的回波。通过改变 gate delay，就可以得到不同距离范围的 gated images。

这篇文献的一个重点是，range-gated image 不是普通二维图像，它的强度和目标距离之间存在关系。在 GRICI 方法中，可以利用至少两张 gated images 的强度关系反推出目标距离。

这对我的研究很关键。因为我现在使用的输入不是直接的深度图，而是多个选通门下的强度图像。也就是说，深度信息不是直接给出的，而是隐藏在不同 gate 的回波强度变化中，需要通过物理建模或者神经网络去利用。

## 第4页：文献1，距离精度与仿真启发

文献1进一步指出，距离精度并不是只由门宽或采样间隔决定的。实际系统中还会受到 shot noise、时间抖动、距离衰减、介质后向散射和光电转换过程的影响。

论文中对比了传统模型和新模型。在 13.45 米的位置，传统模型的相对误差是 76.9%，而新模型只有 3.1%。另外，实验中两个目标的距离差大约达到 7 厘米时，才可以被区分出来。

这对我的仿真有两个启发。第一，Blender 中不能只做非常理想的深度切片，否则会和真实激光选通系统相差比较大。第二，后续数据应该逐步加入门函数、距离衰减、噪声和 gate overlap。

放到真假目标上，真实三维目标应该是不同 gate 出现不同深度结构；平面假目标则应该是完整图案随 gate 响应整体衰减，而不是只渲染出模型的一部分。

## 第5页：文献4，三张 gated images 到深度估计

第二篇重点文献是 Transformer-based 3D range-gated imaging method with multiple depth priors。

这篇和我的工作非常接近，因为它的输入就是三张 gated images，输出是 refined depth map。三张图的系统参数基本相同，只改变激光脉冲和 gate pulse 之间的延时，所以它们覆盖的是不同但有重叠的距离范围。

它的网络骨干改自 AdaBins，同时引入了两个先验：一个是 depth perception prior，也就是近距离分配更密集的深度 bins；另一个是 refined optical depth prior，也就是从选通成像物理关系中提取出来的深度先验。

这篇文献说明了一个很重要的点：多个 gated images 中确实包含可以被学习的深度信息。对我的课题来说，这支撑了我使用 gate stack 而不是单张二维图像来做真假目标识别。

## 第6页：文献4，光学深度先验与网络启发

这篇文献最值得我借鉴的是 optical depth prior 的构造方法。

它根据三张 gated images 中同一像素在不同 gate 下的强度大小，把像素粗略分成近、中、远三个 depth zones。比如某个像素在第一个 gate 中强度最大，就可能属于近距离区域；如果在第二个 gate 中强度最大，就可能属于中间区域。

当然，这个先验也会受到 multipath effect 的影响。论文中提到，反光交通标志等会产生多路径反射，导致某些区域被错误分到远距离区域。因此作者又设计了 refined optical depth prior 来去除这些错误先验。

这对我的启发是：我不仅可以让网络自己学习三个 gate 的特征，也可以显式分析 gate 间强度关系。对于真实三维目标，不同区域可能落在不同 depth zones；而对于平面假目标，大部分区域应该具有更一致的深度响应。

所以后续我的网络可以不仅做普通分类，还可以考虑引入类似 gate consistency 或 depth-response consistency 的约束。

## 第7页：文献2，gated camera 在感知融合中的价值

第三篇文献是 SAMFusion，它发表在 ECCV 2024，研究的是恶劣天气下的三维目标检测。

这篇论文融合了 RGB、gated NIR camera、LiDAR 和 radar。它的背景是，在雾、雪、夜间等条件下，单一传感器很容易失效。例如普通相机受低光影响，LiDAR 在远距离或者恶劣天气下点云会稀疏，而 gated camera 可以通过主动照明和选通机制抑制后向散射，提高低光和恶劣天气下的成像质量。

SAMFusion 的关键不是简单把所有传感器拼接起来，而是根据距离、天气和可见性，自适应地给不同模态分配权重。论文中在远距离行人检测上有明显提升，例如 dense fog 下提升 17.2 AP，heavy snow 下提升 15.62 AP。

这对我的网络设计也有启发。我的三个 gate 虽然不是 RGB、LiDAR、radar 这种不同传感器，但也可以看作三个深度相关的模态。网络应该学习不同 gate 在当前目标和当前距离条件下的可靠性，这就是 gate attention 的意义。

## 第8页：总结与我的研究设计

最后总结一下，这三篇文献分别支撑了我研究中的三个部分。

文献1回答的是物理问题：激光选通回波是怎么形成的，距离精度会受哪些因素影响。它提醒我，仿真不能只做理想切片，而要考虑噪声、抖动、衰减和散射。

文献4回答的是学习问题：三张 gated images 中的强度关系可以形成 optical depth prior，并且可以被 Transformer 类网络用于深度估计。这说明 gate stack 本身是有深度信息的。

文献2回答的是应用和网络融合问题：gated camera 在复杂天气和低光环境下有实际感知价值，而自适应融合机制可以提升鲁棒性。

回到我的研究，我要做的是构建真实三维目标和平面假目标的激光选通图像序列。真实目标在不同 gate 中应该体现出深度结构变化，平面假目标则主要表现为完整平面图案的整体衰减。之后再通过 gate attention 网络，学习这种深度响应差异，实现真假目标识别。

所以我的核心观点是：激光选通的价值不是直接获得一个完美深度图，而是从可实际采集的多门回波强度中学习目标是否具有真实三维结构。
