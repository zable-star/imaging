# 激光距离选通仿真与平面假目标说明

## 1. 当前接收信号定义

当前仿真把一次激光距离选通成像样本表示为三个 gate 图像：

\[
X = [I_1, I_2, I_3]
\]

每个 gate 图像近似表示接收门在不同距离中心处打开时，由目标表面回波形成的二维图像。对真实三维目标，某个表面点的回波强度近似为：

\[
I_g(p) \propto W_g(R_p) \cdot \max(0, n_p \cdot l) \cdot (R_p^2+\epsilon)^{-1} \cdot \exp(-2\sigma R_p)
\]

其中：

- \(R_p\)：表面点到相机/接收端的距离；
- \(W_g(R_p)\)：第 \(g\) 个距离门对该深度的响应；
- \(n_p \cdot l\)：表面朝向与照明方向的 Lambert 项；
- \((R_p^2+\epsilon)^{-1}\)：距离衰减；
- \(\exp(-2\sigma R_p)\)：双程大气衰减。

因此，真 3D 目标的三个 gate 图像不是简单变暗，而是会显示目标不同深度层的表面结构。

## 2. 真目标数据应该是什么样

真目标保留三维几何深度。不同 gate 对不同深度范围敏感：

- gate 1：更偏近距离表面；
- gate 2：中间深度表面；
- gate 3：更远距离表面。

因此真目标的三张图应该呈现空间切片差异：不同部件在不同 gate 中出现或变亮。例如车辆的上表面、车身、发射架或远端结构会因为深度不同而分布在不同 gate 里。

这对应脚本模式：

```text
--target-mode physical-3d
```

## 3. 平面假目标数据应该是什么样

平面假目标不应再表现为三维深度切片。它更合理的物理含义是：攻击者使用一张二维反射图、投影板、平面显示/反射结构或等效平面回波来模拟目标。

因此它的核心特征应该是：

- 三个 gate 中都是同一个完整目标轮廓；
- 轮廓结构不随 gate 改变；
- 亮度只随 gate 响应整体缩放；
- 若平面假目标深度位于 gate 1 附近，则 gate 1 最亮，gate 2/gate 3 逐渐变暗或消失。

可写成：

\[
I_g^{flat}(x,y) = A_g \cdot M(x,y)
\]

其中 \(M(x,y)\) 是同一张二维目标轮廓/反射图，\(A_g\) 是该平面深度处的 gate 响应与距离衰减系数。

这对应脚本模式：

```text
--target-mode flat-echo
```

## 4. 为什么旧 image2d 不合适

旧的 `image2d` 或单帧控制组类似：

```text
[目标图, 黑图, 黑图]
```

这不是严格的平面假目标。它只能说明网络看到“只有一个 gate 有信号”时的退化情况，不能代表真实平面假目标的多 gate 响应。

更合理的平面假目标应该是：

```text
[完整目标轮廓 x 强响应, 完整目标轮廓 x 弱响应, 完整目标轮廓 x 更弱响应]
```

也就是同一张完整二维目标反射图在不同 gate 中以不同强度出现。

## 5. 网络任务解释

当前网络输入是三个 gate 图像。网络需要学习的不是单张图像分类，而是 gate 维度上的回波结构差异：

- 真目标：不同 gate 中的可见结构发生变化，包含三维深度信息；
- 平面假目标：不同 gate 中结构基本相同，只是整体亮度变化，缺少真实深度切片。

因此注意力/融合模块可以理解为在判断：

1. 哪个 gate 的信息最可靠；
2. gate 间是否存在符合三维目标的结构变化；
3. 输入是否只是同一张二维轮廓的强度缩放。

## 5.1 当前网络结构

代码中的主模型是 `SliceAttentionClassifier`。输入张量形状为：

```text
[B, S, C, H, W]
```

其中 \(B\) 是 batch size，\(S=3\) 是 gate 数量，\(C=1\) 是灰度通道，\(H=W=224\)。

模型处理流程为：

1. 每个 gate 图像共用同一个 `SliceEncoder`；
2. `SliceEncoder` 用三层 CNN 提取单 gate 特征；
3. 每个 gate 得到一个 128 维特征向量；
4. attention 分支用 MLP 为每个 gate 预测一个权重；
5. 通过 softmax 得到 gate 权重；
6. 对三个 gate 特征加权求和后分类。

`attention_residual` 模式在 attention 融合之外，还把三个 gate 的特征拼接后经过一层投影，作为残差补回融合特征。它的意义是：

- attention 分支强调最有判别力的 gate；
- residual 分支保留所有 gate 的联合结构；
- 对三维选通切片更稳，因为不会过度依赖单个 gate。

因此，PPT 中不应把当前模型说成 Q/K/V self-attention。更准确的说法是：

```text
shared CNN encoder + gate-wise MLP attention + residual multi-gate fusion
```

如果未来 gate 数量增加到 5/7/9，或把图像切成 patch token，再引入 QKV self-attention 会更合理。

## 6. PPT 推荐表述

可以在汇报中这样讲：

> 对真实三维目标，距离选通成像会在不同接收门中采样目标不同深度处的回波，因此三个 gate 图像具有结构差异。  
> 对平面假目标，由于目标等效位于单一距离平面，三个 gate 中应主要表现为同一二维目标轮廓的整体强度变化，而不是三维结构切片。  
> 因此，本文将真目标建模为 depth-dependent gated echo，将平面假目标建模为 flat-depth gated echo，用于检验网络是否能够利用多 gate 回波一致性识别三维真实性。

## 6.1 可以强调的研究问题

这项工作可以拆成两个层次：

1. 目标类别识别：输入是什么类型的目标，例如坦克、战机、直升机、装甲车、军用卡车；
2. 三维真实性判别：输入是否具有真实三维深度选通结构，还是只是二维平面假目标。

平面假目标可能在单张图像上很像真实目标，因此普通 2D 分类器可能仍会把它识别成“军用卡车”。但多 gate 选通序列提供了额外线索：

- 真 3D：gate 间出现部件级结构变化；
- 平面假目标：gate 间主要是全局亮度变化；
- 因而可以引入“gate consistency / depth variation”作为三维真实性依据。

在 PPT 中可以把这一点表述为：

> 本研究不仅关注目标类别分类，还关注多 gate 回波是否包含真实三维深度结构。平面假目标可以复制目标的二维外形，但难以复制不同深度层在多个接收门中的结构变化。因此，距离选通序列为反假目标识别提供了比单幅图像更强的判别信息。

## 7. 当前需要避免的图示错误

流程图中不要再画：

```text
2D false target = gate 1 有图，gate 2/3 全黑
```

应改成：

```text
2D/flat false target = 三个 gate 均为同一完整目标轮廓，但亮度随 gate 逐渐降低
```

如果为了视觉展示需要，可以设置 `flat-min-response` 为 0.15-0.25，使后两个 gate 仍可见。
