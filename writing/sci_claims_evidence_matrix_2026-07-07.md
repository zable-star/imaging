# SCI claims, evidence, and caveats matrix

This file converts the current experiments into manuscript-safe claims. It is intended to prevent overclaiming and to guide the next manuscript rewrite.

## Core position

The central contribution is:

```text
A physics-interpretable gated-imaging simulation and anti-shortcut validation framework for discriminating true 3D targets from planar false targets.
```

The central contribution is not:

```text
A new neural architecture for military target recognition.
```

## Claims to avoid

Do not write:

```text
The proposed network solves robust military target recognition.
```

Reason: the current dataset is small and simulated.

Do not write:

```text
attention_residual is the best fusion strategy.
```

Reason: four-view strong-noise validation favors simple attention.

Do not write:

```text
The model completely eliminates shortcut learning.
```

Reason: single-gate baselines remain strong under clean/light conditions.

Do not write:

```text
The Blender simulation is equivalent to a real laser gated camera.
```

Reason: the model omits detailed detector response, atmospheric propagation, material BRDF, timing jitter, and real backgrounds.

Do not write:

```text
The current model is robust to realistic reflectance, background, and occlusion changes.
```

Reason: domain-randomized training improves simulated hard-nuisance accuracy, but the dataset is still synthetic, idealized, and small.

## Recommended manuscript thesis

Use this thesis statement:

```text
This work demonstrates that a physically structured gate stack can provide more stable evidence than individual gated images for 3D versus planar false-target discrimination in a controlled Blender simulation, but only after shortcut diagnostics, normalization, grouped validation, and structured-nuisance domain randomization are applied.
```

This is stronger and safer than claiming a new classifier.

## Next evidence needed

1. Broader structured-domain randomization: more reflectance/background/occlusion levels and held-out nuisance types.
2. Harder false-target realism: nonuniform reflectance, small planar deformation, partial occlusion, and simple backgrounds.
3. Broader viewpoint protocol: more yaw angles and at least one elevation change if render time allows.
4. External reference support for military decoys, false targets, and simulation-to-real risk.
5. Final manuscript conversion from working Markdown into a submission-style paper.
